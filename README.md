# GenCallback

## Usage

```
$ mix archive.install github wojtekmach/gen_callback
$ mix gen.callback GenServer
```

Produces the following output on STDOUT:

Since every protocol is also a behaviour, this also works:

```
$ mix gen.callback Enumerable
```
